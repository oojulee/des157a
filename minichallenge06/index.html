Mini Challenge 06

For this challenge do the following

  1. Link the script file in the head of the page.
  2. Be sure to include the defer keyword in the link.
  3. On the script file add the IIFE, add the "use strict" directive, and console.log "running JS".
  4. Then, inside the IIFE write a script that loops through all these list items and turns them green.
  5. Be sure to use const and let to define your variables.
  6. Take a screenshot of the HTML file and the JS file and turn them in.

Hint: You can use a for loop, a for.. of loop, or the forEach array method to loop through the paragraphs. You have examples of each in your files.
